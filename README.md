## Deskripsi
Aplikasi Android sederhana untuk menampilkan form identitas mahasiswa dengan validasi input.

## Fitur
- Form input nama lengkap (EditText)
- Pilihan jenis kelamin (RadioGroup dengan RadioButton)
- Pilihan hobi (CheckBox)
- Validasi input (nama tidak boleh kosong, jenis kelamin harus dipilih)
- Tampilkan hasil input ke TextView
- Pesan error menggunakan Toast dan EditText.error

## Layout yang Digunakan
1. **LinearLayout** - Layout utama dengan orientasi vertikal
2. **LinearLayout** - Layout untuk checkbox hobi dengan orientasi vertikal

## Widget yang Digunakan
- TextView (untuk label dan hasil)
- EditText (untuk input nama)
- RadioGroup dan RadioButton (untuk pilihan jenis kelamin)
- CheckBox (untuk pilihan hobi)
- Button (untuk submit)
- Toast (untuk pesan error)

## Cara Menjalankan
1. Clone repository ini
2. Buka di Android Studio
3. Build and run aplikasi
4. Isi form dan klik tombol "Tampilkan Data" untuk melihat hasil

## Requirements
- Android Studio Arctic Fox atau lebih tinggi
- Android SDK API level 21 atau lebih tinggi
- Kotlin support
